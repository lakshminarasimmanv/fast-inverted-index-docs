Installation Guide

This guide covers the installation of Fast-Inverted-Index for both Python and Rust projects.

Python Installation​

Prerequisites​

Before installing Fast-Inverted-Index, ensure you have:

  • Python 3.7 or later
  • pip (Python package installer)
  • A C++ compiler (for building from source if pre-built wheels are not available)
  • RocksDB (if not using in-memory mode only)

Installing from PyPI​

The simplest way to install is using pip:

pip install fast-inverted-index

Installing with Optional Dependencies​

Install with additional features:

# Install with benchmarking tools
pip install fast-inverted-index[benchmark]

# Install with development tools
pip install fast-inverted-index[dev]

Verifying the Installation​

To verify your installation, run:

import fast_inverted_index
print(fast_inverted_index.__version__)

Rust Installation​

Prerequisites​

Before using Fast-Inverted-Index in your Rust project, ensure you have:

  • Rust 1.58 or later
  • Cargo (Rust package manager)
  • A C++ compiler
  • RocksDB development libraries (if not using in-memory mode only)

Adding to Your Project​

Add Fast-Inverted-Index to your Cargo.toml:

[dependencies]
fast-inverted-index = "0.1.0"

For a specific version or git repository:

[dependencies]
fast-inverted-index = { version = "0.1.0", features = ["python"] }
# OR
fast-inverted-index = { git = "https://github.com/username/fast-inverted-index", branch = "main" }

Enabling Features​

Fast-Inverted-Index provides several feature flags:

[dependencies]
fast-inverted-index = { version = "0.1.0", features = ["python", "rocksdb-fast"] }

Available features:

  • python: Enables Python bindings
  • rocksdb-fast: Optimizes RocksDB for speed over durability
  • memory-optimized: Enables memory optimizations
  • debug-trace: Enables debug tracing (not for production)

Building from Source​

Python Package​

To build the Python package from source:

# Clone the repository
git clone https://github.com/username/fast-inverted-index.git
cd fast-inverted-index

# Install in development mode
pip install -e .

Rust Library​

To build the Rust library from source:

# Clone the repository
git clone https://github.com/username/fast-inverted-index.git
cd fast-inverted-index

# Build the library
cargo build --release

Platform-Specific Instructions​

Linux​

Ubuntu/Debian​

Install the required system dependencies:

# Install RocksDB development libraries
sudo apt update
sudo apt install -y librocksdb-dev libsnappy-dev liblz4-dev libzstd-dev

# Install Python development headers (if building Python bindings)
sudo apt install -y python3-dev

# Install build tools
sudo apt install -y build-essential

CentOS/RHEL/Fedora​

# Install RocksDB development libraries
sudo dnf install -y rocksdb-devel snappy-devel lz4-devel libzstd-devel

# Install Python development headers (if building Python bindings)
sudo dnf install -y python3-devel

# Install build tools
sudo dnf install -y gcc-c++ make

macOS​

Install dependencies using Homebrew:

# Install RocksDB
brew install rocksdb

# Install additional compression libraries
brew install snappy lz4 zstd

# Install Python (if needed)
brew install python

Windows​

Windows support is limited. The recommended approach is using WSL (Windows Subsystem for Linux) or Docker.

For native Windows:

  1. Install Visual Studio with C++ workload
  2. Install RocksDB following RocksDB Windows guide
  3. Set environment variables: 
    set ROCKSDB_LIB_DIR=C:\path\to\rocksdb\lib
    set ROCKSDB_INCLUDE_DIR=C:\path\to\rocksdb\include

Docker Installation​

For containerized deployments, use the provided Dockerfile:

FROM python:3.9-slim

# Install system dependencies
RUN apt-get update && apt-get install -y \
    build-essential \
    librocksdb-dev \
    libsnappy-dev \
    liblz4-dev \
    libzstd-dev \
    && rm -rf /var/lib/apt/lists/*

# Install Fast-Inverted-Index
RUN pip install fast-inverted-index

# Create a volume for persistent storage
VOLUME /data

# Set the working directory
WORKDIR /app

# Copy your application code
COPY . .

# Run your application
CMD ["python", "your_app.py"]

Build and run the Docker container:

docker build -t my-search-app .
docker run -v /path/to/data:/data my-search-app

Troubleshooting​

Common Issues​

Missing RocksDB​

Error: Library not loaded: librocksdbd.dylib or librocksdb.so.X

Solution: Install RocksDB development libraries:

  • Ubuntu/Debian: sudo apt install librocksdb-dev
  • macOS: brew install rocksdb

Compilation Errors​

Error: C++ compilation errors during installation

Solution:

  1. Ensure you have a compatible C++ compiler installed
  2. Install required development libraries
  3. Try installing a pre-built wheel: pip install --only-binary :all: fast-inverted-index

Python Binding Issues​

Error: ImportError: No module named '_fast_inverted_index'

Solution:

  1. Reinstall the package: pip uninstall fast-inverted-index && pip install fast-inverted-index
  2. Ensure you have the correct Python version
