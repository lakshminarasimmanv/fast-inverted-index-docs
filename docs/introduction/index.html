Fast-Inverted-Index

A high-performance search library implemented in Rust with Python bindings.

Fast-Inverted-Index is a production-grade search engine library designed for high performance, memory efficiency, and ease of use. It provides a complete solution for building search functionality into your applications with features like:

  • Full-text search with boolean operators
  • Term autocompletion and suggestions
  • BM25, TF-IDF, and Learning-to-Rank (LTR) algorithms
  • Memory-efficient storage with variable-length integer compression
  • Skip lists for fast query execution
  • Persistent storage using RocksDB
  • Thread-safe LRU caching
  • Comprehensive Python bindings through PyO3

Key Features​

⚡ High Performance​

  • Optimized search algorithms with skip lists
  • Efficient compression with variable-length integer encoding
  • Caching for frequently accessed data
  • Parallel query execution

💾 Flexible Storage​

  • In-memory mode for maximum speed
  • Persistent storage with RocksDB for durability
  • Configurable caching strategy

🔍 Advanced Search Capabilities​

  • Boolean queries (AND, OR)
  • BM25, TF-IDF, and Learning-to-Rank (LTR) ranking
  • Term suggestions and autocompletion
  • Position information for potential phrase queries
  • Field boosting for better relevance

🐍 Python Integration​

  • First-class Python bindings
  • Pythonic API with proper error handling
  • Easy integration with Python applications

🛠️ Robust and Maintainable​

  • Comprehensive test suite
  • Thread-safe implementation
  • Clear documentation
  • Strong error handling

Quick Example​

Python​

from fast_inverted_index import Index

# Create an index
index = Index(in_memory=True)

# Add some documents
index.add_document(1, "Python is a programming language")
index.add_document(2, "Rust is a systems programming language")
index.add_document(3, "Search engines use inverted indices")

# Search for documents
results = index.term_query("programming")
print(f"Found {len(results)} documents containing 'programming'")

# Get term suggestions
suggestions = index.suggest_terms("pro")
for suggestion in suggestions:
    print(f"{suggestion['term']} (score: {suggestion['score']})")

Rust​

use fast_inverted_index::{IndexBuilder, Query};

// Create an index
let mut index = IndexBuilder::new()
    .with_in_memory(true)
    .build()?;

// Add some documents
index.add_document(1, "Rust is a systems programming language")?;
index.add_document(2, "Fast and reliable code with memory safety")?;
index.add_document(3, "Search engines commonly use inverted indices")?;

// Search for documents
let query = Query::term("programming");
let results = index.query(&query)?;

println!("Found {} documents containing 'programming'", results.doc_ids.len());

// Get term suggestions
let suggestions = index.suggest_terms("pro")?;
for suggestion in &suggestions {
    println!("{} (score: {})", suggestion.term, suggestion.score);
}
